Visualizing Linear Regression Model in R

Lizhou Fan

Goal: visualzie the relationship between the topsoil lead concentration (lead column, as y-axis) and the topsoil cadmium concentration (cadmium column, as x-axis).

# previous code
# install.packages("sp")
library(sp)
data(meuse)
m1<-lm(lead~cadmium,data = meuse)
plot(x = meuse$cadmium, y = meuse$lead)
abline(m1)
